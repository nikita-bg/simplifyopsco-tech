Create a new `frontend/src/components/DemoAgent.tsx` component that provides a "Try it now" floating button on the landing page.

**DemoAgent.tsx:**
```
"use client";

import { useState } from "react";
import { useConversation } from "@elevenlabs/react";
import { Mic, MicOff, X, Phone } from "lucide-react";
```

Component behavior:
1. **Floating button** — fixed position `bottom-6 right-6` (z-50), circular (w-14 h-14), gradient bg from primary to indigo-400, with a pulsing ring animation when idle
2. **Label** — small "Try it now" text label next to the button (hidden when panel is open)
3. **On click** — opens a small panel (w-80, glass-card styled) anchored above the button
4. **Panel content:**
   - Header: "Talk to our AI Assistant" with close (X) button
   - Status indicator: shows connection state ("Ready", "Connecting...", "Connected", "Disconnecting...")
   - Mic button: large circular button that calls `startSession({ agentId: process.env.NEXT_PUBLIC_DEMO_AGENT_ID! })` on first press
   - When connected: show audio visualizer (simple CSS bars like the hero section), mic toggle, and end call button
   - When idle: show "Tap to start" prompt
5. **useConversation hook usage:**
   - `const conversation = useConversation();`
   - Start: `await conversation.startSession({ agentId: process.env.NEXT_PUBLIC_DEMO_AGENT_ID! })`
   - End: `await conversation.endSession()`
   - Read: `conversation.status` (idle | connecting | connected), `conversation.isSpeaking`
6. **Error handling:** If `NEXT_PUBLIC_DEMO_AGENT_ID` is not set, do not render the component at all (return null)
7. **Mobile:** On screens < 640px (sm:), make the panel full-width with `fixed inset-x-4 bottom-20`

**In LandingPage.tsx:**
- Import and render `<DemoAgent />` at the bottom of the component, after the footer closing tag but before the fragment closing tag
- Only render if client-side (component is already "use client")

Styling:
- Panel uses `glass-card` utility
- Mic button: `bg-primary hover:bg-primary-hover` when idle, `bg-error hover:bg-error/80` when connected (to indicate "end call")
- Pulsing animation on floating button: use `animate-pulse` ring around it
- Connected state: green dot indicator
