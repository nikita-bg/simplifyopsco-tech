Create a singleton service class following the exact same pattern as `shopify_service.py`. Use `httpx.AsyncClient` with `async with` blocks (same pattern as existing code, not a shared client).

Class `ElevenLabsService` with `BASE_URL = "https://api.elevenlabs.io/v1/convai"`:

1. `_headers() -> dict[str, str]`: Returns `{"xi-api-key": settings.ELEVENLABS_API_KEY, "Content-Type": "application/json"}`

2. `async create_agent(name: str, conversation_config: dict[str, Any], platform_settings: Optional[dict[str, Any]] = None) -> dict[str, Any]`:
   - POST to `{BASE_URL}/agents/create`
   - Payload: `{"name": name, "conversation_config": conversation_config}` + optional platform_settings
   - Timeout: 30s
   - `response.raise_for_status()` then return `response.json()`

3. `async get_agent(agent_id: str) -> dict[str, Any]`:
   - GET to `{BASE_URL}/agents/{agent_id}`
   - Timeout: 30s
   - Return full agent config

4. `async update_agent(agent_id: str, conversation_config: Optional[dict] = None, platform_settings: Optional[dict] = None, name: Optional[str] = None) -> dict[str, Any]`:
   - PATCH to `{BASE_URL}/agents/{agent_id}`
   - Build payload from non-None params only
   - Timeout: 30s

5. `async delete_agent(agent_id: str) -> bool`:
   - DELETE to `{BASE_URL}/agents/{agent_id}`
   - Return `response.status_code == 200`

6. `async get_signed_url(agent_id: str) -> str`:
   - GET to `https://api.elevenlabs.io/v1/convai/conversation/get-signed-url?agent_id={agent_id}`
   - Timeout: 10s
   - Return `data.get("signed_url", "")`

Module-level singleton: `elevenlabs_service = ElevenLabsService()`

Import: `from backend.config import settings` (same pattern as shopify_service.py)
