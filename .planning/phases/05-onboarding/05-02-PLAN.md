<task type="auto">
  <name>Task 2: Create OnboardingProgress component with polling status tracker</name>
  <files>frontend/src/components/OnboardingProgress.tsx</files>
  <action>
Create new `OnboardingProgress.tsx` component that shows real-time onboarding progress.

**Props**: `storeId: string`

**Behavior**:
1. On mount, start polling `GET /api/stores/{storeId}/onboarding-status` every 2 seconds using setInterval
2. Display a vertical step list with 4 visible steps:
   - "Creating your AI agent" (maps to step 'creating_agent')
   - "Setting up knowledge base" (maps to step 'syncing_kb')
   - "Sending welcome email" (maps to step 'sending_email')
   - "Ready to go!" (maps to step 'complete')
3. Each step shows one of three states:
   - **Completed**: Green checkmark icon (Check from lucide-react), text-success color
   - **In progress**: Spinning loader (Loader2 from lucide-react with animate-spin), text-primary color
   - **Pending**: Gray circle outline, text-faint color
4. Determine step state from response:
   - If step is in `completed_steps` array AND is not the current `step` -> completed
   - If step equals current `step` and not is_complete -> in progress
   - If step is in `completed_steps` and `step` equals this step and is_complete -> completed
   - Otherwise -> pending
5. When `is_complete` is true:
   - Stop polling (clearInterval)
   - Show "Your AI assistant is ready!" success message with a green checkmark
   - Wait 1.5 seconds, then call `refetch()` from useStore() to transition to dashboard
6. When `is_failed` is true:
   - Stop polling
   - Show error message from response.error in red
   - Show "Try Again" button that calls refetch() to go back to onboarding form (the store will be recreated)

**Layout**:
- Centered card (max-w-lg) with bg-panel, rounded-2xl, border border-white/5, p-8
- Title: "Setting up your store..." (text-xl font-bold text-heading)
- Subtitle: "This usually takes less than a minute" (text-sm text-muted)
- Steps in a vertical list with left-aligned icons and text
- Each step: flex row with icon (w-6 h-6) + step text (text-sm)
- Spacing between steps: space-y-4 with connecting line (border-l) between icons

**Cleanup**: useEffect cleanup must clearInterval on unmount.

**Import useStore** from `@/lib/store-context` for the refetch function.
**Import apiFetch** from `@/lib/api` for the polling call.
