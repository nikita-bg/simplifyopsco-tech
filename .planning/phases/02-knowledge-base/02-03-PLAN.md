         - Description (textarea, optional)
         - Price (number input, min 0, step 0.01, required)
         - Product URL (text input, optional, placeholder "https://...")
         - Save button (POST to `/api/stores/${storeId}/products`) + Cancel button
       - **Edit**: Clicking edit on a manual product populates the inline form with current values. Save calls PUT.
       - **Delete**: Clicking delete shows a confirmation (simple `window.confirm`). On confirm, calls DELETE endpoint. Removes product from local state.
       - After any CRUD operation, refresh the products list AND refetch KB status (since char count changes).

    4. **Character Limit Warning** (conditional, shown when is_warning=true):
       - Yellow/orange banner below sync status card: "Your knowledge base is approaching the 300,000 character limit. Consider removing products with long descriptions or reducing product count."
