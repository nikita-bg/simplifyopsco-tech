## 🆘 TROUBLESHOOTING:

### MCP Servers не се зареждат:
1. Рестартирай Claude Code
2. Провери `~/.claude/mcp.json` за грешки
3. Help → View Logs за error messages

### Neon не работи:
```bash
npx neonctl@latest me  # Провери authentication
npx neonctl@latest projects list  # Провери projects
```

### GitHub token изтекъл:
```bash
gh auth login  # Re-authenticate
gh auth status  # Провери статус
```

### Firecrawl rate limiting:
- Free tier: 500 credits/month
- Upgrade ако трябва повече: https://www.firecrawl.dev/pricing
